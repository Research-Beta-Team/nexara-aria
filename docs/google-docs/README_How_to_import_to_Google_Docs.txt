How to turn these files into Google Docs
=========================================

These HTML files can be imported into Google Docs so you can edit, share, and present them like normal Docs.

Method 1 — Upload to Google Drive and open as Doc (recommended)
---------------------------------------------------------------
1. Go to https://drive.google.com
2. Click "New" → "File upload"
3. Select one or more .html files from this folder (e.g. CTO_Board_Brief.html, Workflow_Examples_Human_Nexara_ARIA.html, Documentation_Index.html)
4. After upload, right‑click the file → "Open with" → "Google Docs"
5. Google will convert the HTML to a Doc. You can then rename it, move it to a folder, or share it as usual.

Method 2 — Copy-paste from a browser
------------------------------------
1. Double‑click an .html file to open it in your browser
2. Select all (Ctrl+A / Cmd+A), then copy (Ctrl+C / Cmd+C)
3. Go to https://docs.google.com → New document
4. Paste (Ctrl+V / Cmd+V). Tables and headings should come through; you may need to fix a few formats.
